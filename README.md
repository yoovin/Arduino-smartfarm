# Arduino-smartfarm
## 사용부품
- Nodemcu v3
- 배수모터
- 식물 생장 LED
- 4ch 릴레이모듈
- 토양 수분 센서
- RHT03

## 파일
- smartfarm.ino
    아두이노에 컴파일 해 사용하는 파일
- main.html
    smartfarm.ino에 들어가는 html 구문 원본 파일입니다.
